# Система за детектиране на мрежови аномалии

Python проект за симулирање на мрежен трафик и детектиране на аномалии като DDoS поведение, сканиране на портове и необичайни размери на пакети.

## Какво прави

- Симулира нормален и атакуващ трафик
- Детектира аномалии с логика на правилата и анализ на валтаващ Z-score
- Записва събития в файл
- Съхранява пакети, аномалии и устройства в SQLite
- Осигурява богат терминален меню за симулация, ръчен внос, зареждане на файлове и отчети

## Оформление на проекта

- `models/` - основни dataclasses
- `events/` - издаване и абониране на събития
- `services/` - детектиране, анализ, регистриране, постоянство и зареждане на пакети
- `simulation/` - генериране на трафик
- `ui/` - интерфейс на терминала, пряк поток и отчети
- `data/` - редактируеми примерни входни файлове
- `docs/` - архитектура, планиране, тестване и ръководства по операции

## Стартиране

```bash
python main.py
```

Ако терминалът е интерактивен, менюто се отваря. При неинтерактивни стартирания приложението изпълнява демо по подразбиране и излиза.

## Генерирани файлове

- `log.txt`
- `anomalies.db`

## Ръчен вход

Редактируемите примерни пакети живеят в `data/`. Вижте `docs/manual-data/` за пошагови инструкции по зареждане на пакети от менюто или от файлове.
